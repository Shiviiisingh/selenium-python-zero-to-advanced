# selenium-python-zero-to-advanced
A complete beginner-to-advanced journey for learning automation testing using Python and Selenium, including practical examples, framework design, interview preparation, and real-world automation concepts.

- python should be installed in the system
- selenium is required (pip install selenium)

# requirements.txt
It contains all the required library
